Hospedagem github

para criar uma conta no Github.com devemos ter uma conta valida e um repositório com o arquivo index.html

Hospedando um site

  1. Criar um repositório
  2. Publicar o repositório utilizando o comando Publish
  3. Abrir o repositório no VSCode
  4. Criar o index.html
  5. Criar a veesão utilizando o comando Commit
  6. Enviar para o repositório utilizando o comando Push
  7. Entrar no Github.com, acessar o repositório -->settings -->pages -->Escolher a branch source

Branches

Branches são ramificações do seu projeto.
O projeto principal em geral fica na Branche main
Na branche main deve ter sua versão final e funcional de seu projeto
O projeto pode ter branches secundarias
